# Merkle Drupal & Acquia Blog
### Merkle developers blog about Drupal and Acquia.

## Install
[Install Hugo](https://gohugo.io/installation/)

```
git clone git@github.com:merkle-open/drupal-blog.git
cd drupal-blog
hugo server
```
